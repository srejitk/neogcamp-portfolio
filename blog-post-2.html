Dummy Blog #2

August 15th 2021

Can anyone learn to code?

The Idea of coding can get really intimidating especially with the way it's shown in the movies but an hour scoring through YT and a few weeks effort and everyone can understand the beautiful world of coding.

And that is mainly thanks to the abundant amount of resources which learner have access to, from beginners to experts there's always room for everyone, but this abundance comes with a curse.

The Tutorial Hell!! It's easy to binge the learn to code videos in one go and think that you know everything, the truth couldn't be any further from it.

The best way to learn to code is to , you guessed it, code. Consistently. Persistently. It'll be hard, trust me, I face the struggle. But showing up everyday will only make you better.

So see you fellow coder.
